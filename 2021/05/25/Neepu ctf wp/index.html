python pyinstxtractor.py [filename]

得到一个文件夹，里面有一个retest.pyc

反编译这个pyc，得到的py文件里面就有flag，命令

uncompyle6 -o  C:\Users\hp\Downloads\xxx.py C:\Users\hp\Downloads\xxx.pyc
